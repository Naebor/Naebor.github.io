Frequently Asked Questions

I am falling right through the books in Bookin' It sometimes. Is this a bug?

No, this is intended behavior. This behavior is also described in the Information Menu in the game. There is a small sweet spot on each book. If you miss this sweet spot, you picked the incorrect trajectory and will fall through the book. One trajectory is always guaranteed to get to the next book, so at no point is the game impossible.

What are the differences between Classic(3), Classic(4), and Classic(5) in Bookin' It? How do you navigate between these different game modes?

Classic(3) has 3 different launch trajectories. Classic(4) has 4 different launch trajectories. Classic(5) has 5 different launch trajectories. Press the number button inside the Classic game mode button to navigate between Classic(3), Classic(4), and Classic(5). Press anywhere else inside the Classic game mode button to start a game.
